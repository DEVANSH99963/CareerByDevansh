# CareerByDevansh - MERN Placement Portal

## Overview

CareerByDevansh is a full-stack Placement Portal developed using the MERN Stack. It allows recruiters to post jobs and students to search and apply for jobs through a modern and responsive web interface.

## Features

### Authentication
- User Registration
- User Login
- JWT Authentication
- Role-Based Access Control

### Recruiter
- Create Job
- View Own Jobs
- Delete Jobs

### Student
- Browse Available Jobs
- Search Jobs
- Apply for Jobs
- View Applied Jobs

### Profile
- View User Profile
- Logout Functionality

## Tech Stack

### Frontend
- React.js
- Axios
- CSS3

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

### Authentication
- JWT (JSON Web Token)

## Installation

### Clone Repository

```bash
git clone https://github.com/DEVANSH99963/CareerByDevansh.git
```

### Backend Setup

```bash
cd backend
npm install
npm start
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
